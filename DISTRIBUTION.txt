SOUNDGIF 0.1.2
==============

Open soundgif-player (soundgif-player.exe on Windows) to start the desktop app.

Video conversion requires FFmpeg. SoundGIF checks for ffmpeg beside the player,
then checks PATH.

Windows:
  The full Windows release includes ffmpeg.exe.

macOS:
  Install FFmpeg with `brew install ffmpeg`.
  The app is not code-signed. macOS may require you to approve it in
  System Settings > Privacy & Security after the first launch.

Linux:
  Install FFmpeg and the WebKitGTK 4.1 runtime through your package manager.

The command-line tool is named soundgif. Run `soundgif help` for commands.
